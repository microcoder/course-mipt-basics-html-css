Что же выбрать, классы или идентификаторы? id vs class?
Мы выявили для себя следующие закономерности:

  • class - чаще всего используются для визуального стиля
  • id - для логических или для функциональных элементов (связка for-id у label-input)
Есть еще и другая причина, которыю вы видели до этого. Мы не можем навесить элементу несколько идентификаторов. И если мы хотим смиксовать свойства элемента, то мы не сможем этого сделать посредством идентификаторов.
