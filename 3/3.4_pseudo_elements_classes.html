Динамические псевдоклассы

Позволяют выбрать элементы определенного состояния

Еще динамические псевдоклассы:

  • :enabled - Позволяет выбрать все элементы форм в которых есть атрибут enabled
  • :disabled - Позволяет выбрать все элементы форм в которых есть атрибут disabled
  • :checked - Позволяет выбрать все элементы форм в которых есть атрибут checked
  • :indeterminate - Такого состояния нет в HTML, но такое состояние существует у checkbox'ов. Это состояние можно задать из JS. Чаще всего оно употребляется для вложенных списков для того, чтобы выбрать одним checkbox'ом всю категорию. Если вдруг, внутри категории вы выбераете не все элементы, а только некоторые, то селектор для категорий переходит в неопределенное состояние. Он как бы показывает, что у вас в категории есть и выбранные, и не выбранные элементы.
  • :read-only - Позволяет выбрать те элементы у которых задан атрибут только для чтения
  • :valid - Позволяет у элементов форм показать что введенный код, например, валиден.

Структурные псевдоклассы

Позволяют выбрать элементы в зависимости от их положения в структуре HTML документа:

  • :root - позволяет выбрать корневой элемент HTML документа
  • :first-child и :last-child - позволяют выбирать первого или последнего ребенка
  • :nth-child и :nth-last-child(2n+1) - позволяют выбирать n-ого ребенка или n-ого ребенка с конца. В примере выбираются все нечетные дети
  • :nth-of-type и :nth-last-of-type(-n+4) - позволяют выбирать детей не только в зависимости от их положения относительно других детей, но еще предварительно сгруппировав их по типу
  • :only-child и :only-of-type - позволяют выбирать уникальных детей
  • :empty - позволяет выбрать элементы которые пусты, т.е. не содержат никакого контента

Псевдоэлементы

Позволяют выбрать несуществующие элементы на странице HTML документа:

  • ::first-letter { color: red; } - Выбирает первый символ
  • ::first-line { color: blue; } - Выбирает первую строку

Яндекс.Браузер предостерегает пользователей,
когда они начинают вводить пароль на подозрительных страницах.

Существуют еще такие псевдоэлементы как ::before и ::after которые вставляются внутри HTML документа виртуально перед открывающим тегом и перед закрывающим тегом элемента. Они всегда находятся здесь:

<div>
::beforeОднажды, в студеную зимнюю пору
Я из лесу вышел; был сильный мороз.
Гляжу, поднимается медленно в гору
Лошадка, везущая хворосту воз.::after
</div>

Для чего это может быть нужно? Пример:

Однажды, в студеную зимнюю пору
Я из лесу вышел; был сильный мороз.
Гляжу, поднимается медленно в гору
Лошадка, везущая хворосту воз.
