Преобразования элементов

На текущий момент нам доступны 4 вида преобразования:

Перемещение
Масштабирование
Вращение
Наклон

Свойство transform

Для того, чтобы использовать перемещение, нужно использовать свойство transform и в качестве значения указываеть тип трансформации:

.box { transform: тип_трансформации(значение); }

У трансформации есть одна важная особенность, при применении трансформации происходит изменение объекта, но физически объект находится в этом же месте. Например, у нас есть перемещение объекта, но это никак не влияет на окружающие элементы:

Перемещение

Перемещение объекта никак не влияет на окружающие элементы:

transform from
box-1

Еще примеры:

Для демонстрации перемещения, нажмите и задержите клик на каждом из блоков
transform from
translate(-200px, -100px)
translate(200px, 0px);
translate(0px, 100px);
translate(200px, 100px);

Значения также можно задавать в процентах, в этом случае перемещение будет расчитываться относительно размеров блоков, то есть, перемещение по оси x считается относительно ширины, перемещение по оси y в процентах, считается от высоты.

transform from
translate(50%, 50%)

Так же, возможно указать перемещение только по одной из осей указав, например: translateX(50%) или translateY(50%) для того, чтобы не писать нулевые значения для второй оси.

Масштабирование

Масштабирование элемента указывается без каких-либо единиц измерения. По умолчанию значение равно единице, если указывать значение больше единицы, то элемент увеличивается, если меньше, то элемент уменьшается.

box-7
scale(1.5)
scale(0.7)

У функции scale() можно указать как один аргумент, так и два. Если мы указываем 2 аргумента с разными значениями, то сжатие или наоборот, растяжение происходит по оси x и по оси y по разному:

box-7
scale(1, 1.5)
scale(1.5, 1)

Также, в функции scale() можно указывать отрицательные значения. В этом случае происходит переворот элемента:

scale(1, -1)
scale(-1, 1)
scale(-2)

Вращение

Вращение задается при помощи функции rotate(), в качестве аргумента передается угол на который нужно повернуть элемент. Если указывать положительное значение, то элемент поворачивается по часовой стрелке, и наоборот, если указать отрицательное значение, то вращение элемента будет против часовой стрелки:

rotate(30deg)
rotate(-30deg)

Также, значения можно указывать в других единицах измерениях, такие как количество поворотов (turn), градусы (grad) или радианы (rad):

rotate(1.5turn)
rotate(600grad)
rotate(9.42478rad)

Наклон

Наклон выполняется при помощи функции skew(), в нее передается 2 аргумента, наклон по оси X и наклон по оси Y:

skew(25deg, 0)
skew(0, 25deg)
skew(25deg, 25deg)
